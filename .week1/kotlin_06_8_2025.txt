Kotlin Programming:
---------------------------------------------------------------------------------------------------------------------------------------------

Kotlin is a high level programming language mainly used for the development of android applications.
-It was developed by Google (jetBrains corporation).
-Kotlin was created as an alternate to java PL.
-Earlier they use java as the primary language for android applications development but for some reason google developed their language for android applications and it is called as kotlin.
